# rn-stylish-progress-bar

Stylish progress bar for React Native

## Installation

```sh
npm install rn-stylish-progress-bar
```

## Usage

```js
import StylishProgressBar from 'rn-stylish-progress-bar';

// ...

<StylishProgressBar min={min} max={10} />;
```
