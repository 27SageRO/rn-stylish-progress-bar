# rn-stylish-progress-bar

Stylish progress bar for React Native

## Installation

```sh
npm install rn-stylish-progress-bar
```

## Usage

```js
import { multiply } from "rn-stylish-progress-bar";

// ...

const result = await multiply(3, 7);
```
